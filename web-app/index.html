场景案例

真实场景操作演示，快速上手使用

使用 WPS 插件生成汇报材料

场景：需要根据要点生成正式汇报材料。例如：本季度完成项目A、进度80%、预计下季度完成、存在资源不足风险。

操作步骤

  1. 打开 WPS Office，新建空白文档
  2. 在菜单栏找到「开悟插件」选项卡
  3. 点击「帮我写」按钮
  4. 输入主题或要点
  5. AI 根据输入生成汇报材料
  6. 审核内容，按需润色、扩写或缩写
WPS插件可调用 可生成正文 可润色 可扩写/缩写 可生成标题

使用本地搜索查找项目资料

场景：查找最近一个月修改过的「智能办公」相关材料。

操作步骤

  1. 打开 Everything（快捷键 Ctrl+Shift+F）
  2. 在搜索框输入关键词「智能办公」
  3. 点击筛选按钮，选择「修改时间」→「最近一个月」
  4. 浏览搜索结果，双击文件直接打开
返回文件列表 可打开文件 可复制路径 可生成材料清单

使用 OCR 识别扫描件并入库

场景：识别一张扫描图片，保存到个人知识库。

操作步骤

  1. 准备清晰可读的图片
  2. 启动 OCR 识别工具
  3. 选择图片，选择识别语言（中文/英文/中英混合）
  4. 点击「开始识别」
  5. 点击「保存到知识库」，选择 Obsidian 目录
调用 Tesseract 输出 TXT/Markdown 写入 Obsidian Vault 可被 Everything 检索

使用 Obsidian 沉淀项目知识

场景：把一份材料整理成项目知识卡片。

操作步骤

  1. 启动 Obsidian，打开 Vault：KaiwuVault
  2. 选择保存目录（01_项目资料 / 04_个人知识）
  3. 新建笔记，输入标题
  4. 使用 Markdown 编写内容
  5. 添加标签和关联链接
自动生成 Markdown 按模板归档 生成标签 生成关联链接

使用 Agent 分析安装日志

场景：WPS 插件加载失败，分析安装日志找出原因。

操作步骤

  1. 打开日志目录：logs/
  2. 查找安装日志 install_*.log
  3. 搜索关键词：WPS、插件、错误、失败
  4. 查看错误代码和描述
  5. 根据错误信息查找解决方案或运行 repair.bat
读取日志 分析错误 给出修复建议 可调用 repair 脚本

使用 XMind 生成方案结构

场景：为新项目创建方案构思思维导图。

操作步骤

  1. 启动 XMind
  2. 选择「方案构思模板」
  3. 输入项目名称作为中心主题
  4. 添加一级分支（目标、范围、资源、风险等）
  5. 为每个分支添加子主题
  6. 导出为图片或 PDF
可打开模板 可编辑内容 可导出文件 可保存项目

使用说明

简单四步，即可开始智能办公

1

安装前准备

  • Windows 7 SP1 64位
  • 至少 4GB 内存
  • 至少 10GB 磁盘
  • 管理员权限
2

一键安装

  • 右键点击 install.bat
  • 选择「管理员身份运行」
  • 按提示完成安装
  • 安装后建议重启
3

开始使用

  • 桌面快捷方式启动
  • WPS 中使用开悟插件
  • Ctrl+Shift+F 快速搜索
  • OCR 一键识别入库
4

维护管理

  • check.bat 一键检测
  • repair.bat 一键修复
  • uninstall.bat 卸载
  • 日志自动记录

常见问题 FAQ

遇到问题？这里可能有你要的答案

  • 确保以管理员身份运行安装脚本
  • 检查系统是否为 Windows 7 SP1 64位
  • 确保磁盘空间充足（至少 10GB）
  • 查看安装日志：logs/install_*.log
  • 检查 WPS 版本是否兼容
  • 重新安装 WPS 插件
  • 重启 WPS Office
  • 运行 repair.bat 修复插件
  • 确保 Everything 服务正在运行
  • 尝试重建索引
  • 检查搜索路径设置
  • 重启 Everything 服务
  • 使用更清晰的图片
  • 选择正确的识别语言
  • 调整图片方向
  • 确保图片分辨率足够
  • 安装日志：logs/install_*.log
  • 检测报告：reports/check_report_*.md
  • 下载日志：logs/download_*.log
  • 运行 uninstall.bat 一键卸载
  • 卸载不会删除个人文档和知识库
  • 如需删除知识库请手动操作
