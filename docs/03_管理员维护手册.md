# V1.4.1 管理员维护手册

- 安装前：验证 ZIP、14 项清单文件和 KexStepup 制品来源。
- 安装后：重启并运行 `check.bat`；当前检测脚本包含 20 项健康检查和 WPS/PPT 门禁。
- 修复：运行 `repair.bat`，一次只选择一个修复项并复核结果。
- 卸载：运行 `uninstall.bat`；用户 Obsidian Vault 默认保留。
- 日志：集中保存 `runtime/logs/` 和 `runtime/reports/`，不得将用户名、路径或业务内容上传到未经批准的平台。
- 更新：修改版本清单、重新校验、跑回归测试、完成 Win7 实机测试后才能发布。

WPS/Kaiwu 与 PPT workflow 当前不是已交付能力。只有通过专项验收后，才允许把对应能力从 unsupported/待验证改为可用。

当前没有自动回滚事务；安装失败应按日志逐项卸载已安装组件。

- KexStepup：取得安装包后先运行 `scripts/admin/compute_kexstepup_hash.bat`，按报告和审批记录更新清单。
- 发布前：先运行 `release_preflight.bat`，再运行 `tests/run_tests.ps1` 和 `build_zip.bat`。
