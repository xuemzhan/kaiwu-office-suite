# Win7 SP1 验证清单 (Test Plan)

> 本文件提供给 Win7 SP1 实机/VM 验证人员使用
> **目标**: 在 Windows 7 SP1 64 位 / PowerShell 5.1 环境下跑通 39 项回归测试
> **当前基线 (Win10)**: 39 PASS / 0 FAIL / 100% (TEST_REPORT.md)

---

## 1. 环境准备

### 1.1 系统要求
- [ ] Windows 7 SP1 64 位 (Build 7601.x)
- [ ] PowerShell 5.1 (`$PSVersionTable.PSVersion`)
- [ ] 管理员账户
- [ ] 至少 10GB 磁盘空间
- [ ] 已安装的运行时:
  - [ ] .NET Framework 4.8
  - [ ] WebView2 Runtime 109
  - [ ] VC++ Runtime
  - [ ] Git for Windows 2.46.2 (`git --version`)
  - [ ] Node.js (for OpenCode 工具, 非必需)

### 1.2 套件部署
1. 复制 `kaiwu-office-suite-v1.3.3.zip` 到 Win7 主机
2. 解压到 `C:\KaiwuOfficeSuite\` (或自定义路径, 后续用环境变量覆盖)
3. 确认 ZIP SHA256 与 `.sha256` 文件匹配:
   ```cmd
   certutil -hashfile kaiwu-office-suite-v1.3.3.zip SHA256
   ```
4. 验证安装器 SHA256:
   ```cmd
   cd C:\KaiwuOfficeSuite
   for /F "tokens=*" %i in (manifest\SHA256SUMS.txt) do @echo %i
   ```

## 2. 一键安装

右键 `install.bat`, 选择 "以管理员身份运行", 等待 15-30 分钟.

完成后执行 `check.bat` 确认所有组件 PASS.

## 3. 跑回归测试 (39 项)

### 3.1 执行命令
```cmd
cd C:\KaiwuOfficeSuite
powershell -ExecutionPolicy Bypass -File tests\run_tests.ps1
```

### 3.2 预期结果
- **目标**: 39/39 PASS (100%) — 套件设计基线

- **任何 [PASS] 数量低于基线**即为回归

### 3.3 测试报告
测试结束自动生成 `tests\TEST_REPORT.md`, 复制此文件回到工作区即可对比.
