# V1.4.1 场景案例手册

## 已验证脚本场景

1. Everything/PowerShell 本地文件名检索。
2. Tesseract 图片 OCR 转 Markdown。
3. OCR 结果写入 Obsidian Vault。
4. Git 仓库状态报告。
5. Obsidian Markdown 笔记创建。

## WPS 与 PPT 场景边界

- WPS AI 插件来源参考 `xuemzhan/Kaiwu`。上游能力覆盖写作、润色、翻译、总结、文档问答、思维导图等方向，但上游要求 Windows 10/11 与 WPS 12.1.0.26375+；V1.4.1 目标为 Windows 7 SP1，当前不能作为已交付能力演示。
- PPT 能力参考 `xuemzhan/ppt-workflow`。本仓已完成源码层面初审，确认其为 Python/CLI 流水线且要求 Python 3.12+；当前仅提供 unsupported 门禁脚本，尚未完成 Win7 移植或实机验收，不能声明 PPT 自动排版/生成已可用。

## 未完成场景

WPS 智能总结、云端模型调用、PPT 自动生成/排版、MCP/A2A 服务、会议语音转写、自动任务分配和语义文件检索均未完成端到端验收，不应作为当前版本演示承诺。
