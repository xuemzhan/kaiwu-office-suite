# 开悟个体增智智能办公套件 V1.0 用户使用手册

**版本:** V1.0  
**发布日期:** 2026-06-17  
**目标环境:** Windows 7 SP1 64位  
**目标用户:** 普通办公用户

---

## 一、套件概述

开悟个体增智智能办公套件 V1.0 是一套面向Windows 7 SP1 64位办公环境的智能办公解决方案。本套件整合了多种办公工具，提供智能办公、本地搜索、OCR识别、知识库管理等功能。

### 核心功能

1. **智能办公** - WPS插件提供AI辅助写作
2. **本地搜索** - Everything快速搜索本地文件
3. **OCR识别** - Tesseract识别图片文字
4. **知识库管理** - Obsidian管理个人知识
5. **思维导图** - XMind创建思维导图

---

## 二、快速开始

### 2.1 启动套件

安装完成后，您可以通过以下方式启动：

1. **桌面快捷方式** - 双击桌面上的"开悟办公套件"图标
2. **开始菜单** - 点击开始菜单中的"开悟办公套件"程序组
3. **任务栏** - 固定常用工具到任务栏

### 2.2 主要入口

| 入口 | 功能 | 快捷键 |
|------|------|--------|
| WPS智能写作 | AI辅助写作 | 无 |
| 本地文件检索 | 搜索本地文件 | Ctrl+Shift+F |
| OCR文字识别 | 识别图片文字 | 无 |
| 知识库管理 | 管理个人知识 | 无 |

## 三、WPS智能写作

### 3.1 启动WPS插件

1. 打开WPS Office
2. 在菜单栏找到"开悟插件"选项卡
3. 点击需要的插件功能

### 3.2 主要功能

#### 帮我写

1. 在WPS中输入主题或要点
2. 点击"帮我写"按钮
3. AI将根据输入生成内容
4. 可选择生成正式汇报、工作计划等格式

#### 帮我改

1. 选中需要修改的文本
2. 点击"帮我改"按钮
3. AI将提供修改建议
4. 可选择接受或拒绝建议

#### 帮我润色

1. 选中需要润色的文本
2. 点击"帮我润色"按钮
3. AI将优化文本表达
4. 使文本更加流畅、专业

#### 帮我总结

1. 选中需要总结的文本
2. 点击"帮我总结"按钮
3. AI将提取关键信息
4. 生成简洁的摘要

## 四、本地文件检索

### 4.1 启动Everything

1. 双击桌面上的"Everything"图标
2. 或使用快捷键 `Ctrl+Shift+F`

### 4.2 搜索文件

#### 基本搜索

1. 在搜索框输入关键词
2. 实时显示搜索结果
3. 双击文件可直接打开

#### 高级搜索

1. **按文件类型搜索**：`ext:docx` 或 `ext:pdf`
2. **按路径搜索**：`path:documents`
3. **按大小搜索**：`size:>10MB`
4. **按日期搜索**：`dm:today`（今天修改）

---

## 五、OCR文字识别

### 5.1 启动OCR工具

1. 双击桌面上的"OCR识别"图标
2. 或使用命令行调用

### 5.2 识别图片

1. **选择图片** - 点击"选择图片"按钮
2. **选择语言** - 选择识别语言（中文/英文/中英混合）
3. **开始识别** - 点击"开始识别"按钮
4. **查看结果** - 识别结果将显示在文本框中

---

## 六、知识库管理

### 6.1 启动Obsidian

1. 双击桌面上的"Obsidian"图标
2. 打开Vault：`KaiwuVault`

### 6.2 知识库结构

```
KaiwuVault/

├── 01_项目资料/       # 项目相关资料

├── 03_OCR识别/        # OCR识别结果
├── 04_个人知识/       # 个人知识积累
├── 05_模板库/         # 笔记模板
└── 99_归档/           # 归档资料
```

## 七、常见问题

### 1. WPS插件不显示怎么办？

1. 检查WPS版本是否兼容
2. 重新安装WPS插件
3. 重启WPS Office

### 2. Everything搜不到文件怎么办？

1. 检查Everything服务是否运行
2. 重建索引
3. 检查搜索路径设置

### 3. OCR识别效果差怎么办？

1. 使用更清晰的图片
2. 选择正确的语言
3. 调整图片方向
