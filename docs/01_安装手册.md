# V1.4.1 安装手册

## 前置条件

Windows 7 SP1 64 位、管理员权限、4 GB 内存和 10 GB 磁盘空间。确认所有内部安装包来源已审批。

## 步骤

1. 校验交付 ZIP 的 SHA256。
2. 解压到 `C:\KaiwuOfficeSuite`。
3. 管理员运行 `verify_installers.bat`。缺失或不匹配时禁止继续。
4. 管理员运行 `install.bat`。
5. 按提示重启系统。
6. 运行 `check.bat`，保存 `runtime/logs/` 和 `runtime/reports/` 中的日志与 Markdown 报告。

`check.bat` 会执行 20 项健康检查。WPS 总结和 PPT workflow 当前是 unsupported 门禁：返回预期退出码 2 表示门禁正常，不表示真实 AI 总结或 PPT 生成已交付。

KexStepup 是 Win7 必装兼容组件；缺失时安装器和打包器都会退出 1。XMind 是可选组件。

## 故障处理

先查看 `runtime/logs/`、`runtime/reports/` 和 `runtime/state/install_state.json`。禁止通过修改清单哈希来掩盖来源不明或损坏的安装包。
