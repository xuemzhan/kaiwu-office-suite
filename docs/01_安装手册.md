## 一、安装前准备

### 1. 系统要求

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 7 SP1 64位 | Windows 7 SP1 64位 |
| 处理器 | 1GHz | 2GHz或更高 |
| 内存 | 2GB | 4GB或更高 |
| 硬盘空间 | 10GB可用空间 | 20GB可用空间 |
| 网络 | 无需网络（离线安装） | 无需网络 |
| 权限 | 管理员权限 | 管理员权限 |

### 2. 准备工作

1. **备份重要数据** - 安装前建议备份重要文件
2. **关闭杀毒软件** - 临时关闭Windows Defender或其他杀毒软件
3. **检查磁盘空间** - 确保安装目录有足够空间
4. **以管理员身份运行** - 右键点击安装程序，选择"以管理员身份运行"

### 3. 文件准备

确保已获取以下文件：
- `kaiwu-office-suite-v1.0.zip` - 套件压缩包
- `SHA256SUMS.txt` - 哈希校验清单
- `install.bat` - 一键安装脚本

## 二、安装步骤

### 步骤1：解压套件

1. 将 `kaiwu-office-suite-v1.0.zip` 解压到目标目录
2. 建议解压到 `C:\KaiwuOfficeSuite` 或 `D:\KaiwuOfficeSuite`
3. 确保解压路径不含中文或特殊字符

### 步骤2：校验文件完整性

1. 打开命令提示符（管理员）
2. 进入解压目录
3. 运行以下命令校验：

```cmd
certutil -hashfile "kaiwu-office-suite-v1.0.zip" SHA256
```

4. 将输出的哈希值与 `SHA256SUMS.txt` 中的值对比
5. 确保哈希值一致

### 步骤3：运行安装程序

1. 右键点击 `install.bat`
2. 选择"以管理员身份运行"
3. 按照提示完成安装
4. 安装完成后重启计算机（建议）

### 步骤4：验证安装

1. 运行 `check.bat` 检测安装状态
2. 查看检测报告 `reports/check_report_*.md`
3. 确保所有组件显示"已安装"

## 四、管理员权限说明

### 为什么需要管理员权限？

1. **安装系统组件** - .NET Framework、VC++ Runtime等需要管理员权限
2. **修改系统配置** - 环境变量、注册表等需要管理员权限
3. **创建系统服务** - Everything等服务需要管理员权限

### 如何以管理员身份运行？

1. 右键点击 `install.bat`
2. 选择"以管理员身份运行"
3. 在UAC提示中点击"是"

---

## 五、重启说明

### 何时需要重启？

1. **.NET Framework 4.8 安装后** - 可能需要重启
2. **VC++ Runtime 安装后** - 可能需要重启
3. **WebView2 Runtime 安装后** - 可能需要重启
4. **WPS Office 安装后** - 可能需要重启

### 重启策略

安装程序不会强制重启，而是：
1. 记录需要重启的状态
2. 提示用户重启
3. 重启后可继续执行安装
4. 支持断点续装

## 六、安装完成检查

### 检查清单

1. [ ] 运行 `check.bat`，所有组件显示"已安装"
2. [ ] 桌面出现快捷方式
3. [ ] 开始菜单出现程序组
4. [ ] WPS Office可正常启动
5. [ ] Everything服务正在运行
6. [ ] Tesseract-OCR可识别测试图片

### 常见问题

#### 安装失败怎么办？

1. 查看安装日志 `logs/install_*.log`
2. 检查错误信息
3. 根据错误信息采取相应措施
4. 重新运行安装程序

#### 没有管理员权限怎么办？

1. 联系IT管理员获取管理员权限
2. 或使用便携版（如可用）

#### WebView2安装失败怎么办？

1. 检查系统是否满足要求
2. 手动下载安装WebView2 Runtime
3. 重新运行安装程序

## 七、卸载说明

### 卸载方法

1. 运行 `uninstall.bat`
2. 按照提示完成卸载
3. 重启计算机（建议）

### 卸载注意事项

1. 个人文档和知识库不会被删除
2. 快捷方式和配置文件会被清理
3. 某些组件可能需要手动卸载
