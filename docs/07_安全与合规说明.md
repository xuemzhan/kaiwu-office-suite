# 开悟个体增智智能办公套件 V1.0 安全与合规说明

**版本:** V1.0  
**发布日期:** 2026-06-17  
**目标环境:** Windows 7 SP1 64位

---

## 一、下载来源说明

### 1.1 官方来源

所有软件均从以下官方来源获取：

| 软件 | 官方来源 | 验证方式 |
|------|----------|----------|
| .NET Framework 4.8 | 微软官方网站 | 数字签名验证 |
| WebView2 Runtime | 微软官方网站 | 数字签名验证 |
| VC++ Runtime | 微软官方网站 | 数字签名验证 |
| Git for Windows | GitHub官方仓库 | SHA256验证 |
| Everything | voidtools官方网站 | 数字签名验证 |
| Tesseract-OCR | GitHub官方仓库 | SHA256验证 |
| WPS Office | 金山官方网站 | 数字签名验证 |
| Obsidian | obsidian.md官方网站 | 数字签名验证 |

### 1.2 禁止来源

以下来源的软件**禁止使用**：

1. **未知网盘** - 百度网盘、腾讯微云等非官方分享
2. **个人博客** - 非官方技术博客提供的下载链接
3. **破解站** - 提供破解版、注册机的网站
4. **广告下载站** - 包含大量广告的下载网站
5. **第三方捆绑下载站** - 捆绑其他软件的下载站

## 二、哈希校验说明

### 2.1 SHA256校验

所有软件安装包均提供SHA256哈希值，用于验证文件完整性。

**校验方法：**

```cmd
certutil -hashfile "文件名" SHA256
```

**校验文件：**

- `manifest/SHA256SUMS.txt` - 包含所有软件的SHA256哈希值

### 2.2 校验流程

1. **下载前校验** - 下载前验证SHA256
2. **下载后校验** - 下载后验证SHA256
3. **安装前校验** - 安装前验证SHA256
4. **安装后校验** - 安装后验证文件完整性

---

## 三、离线安装说明

### 3.1 离线安装原则

本套件支持完全离线安装，不依赖互联网连接。

**离线安装优势：**

1. **安全性** - 不暴露于网络攻击
2. **可控性** - 完全控制安装过程
3. **可重复性** - 可重复安装
4. **可审计性** - 安装过程可审计

### 3.2 离线安装流程

1. **准备安装包** - 在联网环境中下载所有安装包
2. **验证完整性** - 验证SHA256哈希值
3. **复制到目标机** - 将安装包复制到目标计算机
4. **执行安装** - 运行install.bat进行安装
5. **验证安装** - 运行check.bat验证安装结果

---

## 四、不自动升级说明

### 4.1 升级策略

本套件默认禁止自动升级，所有软件版本锁定。

**升级策略：**

1. **版本锁定** - 所有软件版本锁定
2. **禁止自动升级** - 不允许软件自动升级
3. **手动升级** - 需要管理员手动升级
4. **测试验证** - 升级前必须测试验证

## 五、本地数据存储说明

### 5.1 数据存储位置

所有用户数据默认存储在本地计算机。

**存储位置：**

| 数据类型 | 存储路径 | 说明 |
|----------|----------|------|
| 知识库 | `%USERPROFILE%\Documents\KaiwuVault` | Obsidian知识库 |
| 配置文件 | `config/` | 各组件配置文件 |
| 日志文件 | `logs/` | 系统日志文件 |
| 个人文档 | 用户自定义 | 用户个人文档 |

### 5.2 数据安全

1. **本地存储** - 数据不上传到云端
2. **访问控制** - 限制未授权访问
3. **加密存储** - 敏感数据加密存储
4. **备份支持** - 支持本地备份

## 七、禁止联网调用说明

### 7.1 联网限制

本套件默认禁止联网调用，所有功能在本地运行。

**禁止联网的功能：**

4. **远程访问** - 禁止远程访问
