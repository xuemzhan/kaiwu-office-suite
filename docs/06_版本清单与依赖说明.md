## Win7 兼容性原则

- 安装与校验链优先使用 Windows 7 SP1 自带能力：`cmd`、`certutil`、Windows Script Host。
- PowerShell 集成脚本建议 5.1；安装包 SHA256 校验不依赖 Python。
- 任何上游仅声明 Windows 10/11 的组件，必须先完成回移植方案、兼容层方案或 Win7 实机验收，不能直接列为“已验证”。
