## 二、版本锁定策略

### 2.1 锁定原则

1. **所有软件版本锁定** - 不允许自动升级
2. **WebView2 Runtime固定109系列** - 不使用最新版
3. **Git for Windows固定2.46.2** - 不升级
4. **.NET Framework使用4.8离线安装包** - 不使用Web在线安装包
5. **Electron类应用必须验证Win7兼容性** - 确保兼容性
